ev-pro
======

This folder contains abstractions for projection mapping.

There are currently 5 projection modules to select from:
* [ev_pro_22] has 2x2 cornerpoints for flat surfaces.
* [ev_pro_33] has 3x3 cornerpoints for curved surfaces.
* [ev_pro_55] has 5x5 cornerpoints for heavily distorted surfaces.
* [ev_pro_22easy] is an alternative to ev_pro_22, with less functionality.
* [ev_pro_33easy] is an alternative to ev_pro_33, with less functionality.

Additional modules:
* [ev_texhelper] is a tool for easy texture coordinates selection.
